Adding Registration

To make it easier to add registration to a webpage, JavaScript libraries were created that can be added to your page that will do most of the heavy lifting.

To add side-panel registration, add the registration.js JavaScript file to your webpage:

<script src="../js/registration-forms.js"></script>

You can clone the bigspotteddog/tiles repository, or copy the registration.js file to your own website.

Once registration.js has been added to your webpage, you can give it the container div and it will add the registration form to it.

<script>
  window.addEventListener("load", (event) => {
    const forms = new RegistrationForms().Builder()
      .withNewRegistrationForm("registration-container")
      .create();
  });
</script>

The registration form will appear when the page loads in the container you gave it and it will look something like this:

To add your own form, you can simply add the form elements,

<script>
  window.addEventListener("load", (event) => {
    const forms = new RegistrationForms().Builder()
      .withNewRegistrationForm("registration-container")
      .withNewRegistrationForm("inline-registration-container")
        .withRegistrationNameId("inline-registration-name")
        .withRegistrationEmailId("inline-registration-email")
        .withRegistrationButtonId("inline-registration-button")
        .withManualForm(true)
      .create();
  });
</script>

There are other customizations that can be made as well. See registration.js for the actual code behind the RegistrationForm.
